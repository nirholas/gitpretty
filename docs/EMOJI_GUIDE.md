## Design Principles

The Aesthetics emoji set follows these principles:

1. **No faces** - No emoji with facial expressions
2. **No rainbows** - Clean tech aesthetic
3. **Builder focus** - Tools, tech, geometric shapes
4. **Visual distinction** - Each emoji easily recognizable
5. **Unicode support** - Works in all modern terminals
